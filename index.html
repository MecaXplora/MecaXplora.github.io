Proyectos
Brazo Robótico

Descripción del proyecto de brazo robótico. Este proyecto consiste en diseñar, construir y programar un brazo robótico con capacidad de movimiento y manipulación de objetos.

El brazo estará compuesto por servomotores, sensores y una interfaz de control que permitirá al usuario controlar los movimientos del brazo y realizar tareas específicas.

Robot Seguidor de Línea

Descripción del proyecto de robot seguidor de línea. Este proyecto consiste en diseñar y construir un robot capaz de seguir una línea trazada en el suelo utilizando sensores de infrarrojos o cualquier otro tipo de sensor adecuado.

El robot deberá ser capaz de detectar la línea y ajustar su trayectoria para seguir el recorrido de manera autónoma, utilizando algoritmos de control adecuados.
